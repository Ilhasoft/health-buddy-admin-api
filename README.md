# HealthBuddy

## Getting started

### Set environment variable
Create a file called `.env` at the root of the project and assign the following variables:

- `POSTGRES_DB`
- `POSTGRES_USER`
- `POSTGRES_PASSWORD`
- `POSTGRES_HOST`

- `DEBUG`
- `SECRET_KEY`

- `SENDGRID_API_KEY`

- `AWS_ACCESS_KEY_ID`
- `AWS_SECRET_ACCESS_KEY`
- `AWS_STORAGE_BUCKET_NAME`
- `AWS_S3_REGION_NAME`

- `DEFAULT_FROM_EMAIL`

- `TOKEN_ORG_RAPIDPRO`

### Docker
```shell script
$ docker-compose up --build
```
